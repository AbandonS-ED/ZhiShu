## 环境配置

### 数据库连接

- **PostgreSQL**: `postgresql+asyncpg://postgres:123456@localhost:5432/zhishu`
- **Redis**: `redis://localhost:6379/0`
- **LLM**: MiniMax-M3 (`https://api.minimax.chat/v1`)
- **备选 LLM**: 讯飞星火 V4 (`LLM_PROVIDER=spark`)

### 端口约定

- **后端**: `http://localhost:8001`
- **前端**: `http://localhost:3000`
- **API 文档**: `http://localhost:8001/docs`

### 演示入口

```bash
# 一键启动
.\start.ps1

# 或手动启动
# 后端
cd backend && venv\Scripts\activate
uvicorn app.main:app --reload --host 0.0.0.0 --port 8001

# 前端
cd frontend && npm run dev

# 学生端: http://localhost:3000
# 管理端: http://localhost:3000/admin/login (admin / admin123)
# API 文档: http://localhost:8001/docs
```
