// student_profiles.dimensions · 7 维画像 { "comprehension": { "score": 75, "confidence": 0.80 }, // 理解力 "memory": { "score": 68, "confidence": 0.70 }, // 记忆力 "application": { "score": 82, "confidence": 0.85 }, // 应用转化 "imagination": { "score": 60, "confidence": 0.60 }, // 想象力 "focus": { "score": 78, "confidence": 0.75 }, // 专注力 "knowledge_base": { "score": 70, "confidence": 0.80 }, // 知识基础 "learning_goal": { "score": 85, "confidence": 0.90 } // 学习目标 }

⚡规则引擎（即时）

• tiku 每题 → application（正确率映射）
• zixi 学习 → memory + focus（时长映射）
• 节点完成 → learning_goal（+2/节点）
• 评估报告 → comprehension + knowledge_base
• 错题回顾 → memory（+1/+3/-1）
• 思维导图 → imagination（+1/次）

🧠AI 分析（批量）

• tiku 每 5 题 → 全 7 维 AI 推断
• zixi 结束 → 全 7 维 AI 推断
• resources 学习 → analyzeBehavior('study')
• duihua 对话 → analyzeBehavior('chat')
• 定时分析 → 每 4 小时 scheduled_analysis
• 缓存 5min + 行为后自动清 localStorage

08
F3

个性化学习路径

DAG 拓扑排序 · 节点解锁机制

# 算法思路 1. 获取课程知识图谱 2. 过滤已掌握知识点 （基于画像 knowledge_base） 3. 拓扑排序（Kahn 算法） 确保先修在前 4. 薄弱知识点优先 （基于弱项维度） 5. 分配每日计划 （根据 daily_hours）
① 输入知识点
如"机器学习, CNN, 深度学习"
② AI 生成 10-15 节点路径
learning_path_agent 自主分析拆分
③ 节点学习
AI 对话 + 学习资源双模式
④ AI 测验解锁
≥60 分解锁下一节点
⑤ 综合测试
所有节点完成后综合考察
09
INNOVATION

创新点

错题本 Agent 化 · 自习模式 AI 视觉

📒错题本 · AI 错因分析

4 步思考链：错因分类 → 策略讲解 → 同类题推荐 → 反思重生成

SSE 真流式：每步思考链日志实时输出

5 类错误标签：计算 / 概念 / 审题 / 粗心 / 未分析

掌握度算法：答对 +20% · 上限 100%

自动收录：/tiku 答错即加入错题本

📷自习模式 v1.5

TF.js + MoveNet Lightning：浏览器本地姿态检测

~3MB 模型 · CPU ~30ms/帧 · 零上传

3 个关键点：鼻子 + 左肩 + 右肩

3 档巡查：宽松 120s / 标准 60s / 严格 30s

3 状态机：idle → running → report

联动画像：专注力数据自动更新 focus 维度
