# 智学 (ZhiShu) 前端开发计划

> **版本**: v1.0  
> **日期**: 2026-06-06  
> **技术栈**: Next.js 14 (App Router) + Tailwind CSS + shadcn/ui + Zustand

---

## 1. 项目概述

### 1.1 目标
构建一个多智能体个性化学习资源生成系统的前端界面，支持：
- 对话式学习画像构建 (F1)
- 多智能体资源生成展示 (F2)
- 自适应学习路径可视化 (F3)
- 智能辅导问答 (F4)
- 学习进度追踪 (F5)

### 1.2 技术约束
- **必须使用**: Next.js 14 App Router + Tailwind + shadcn/ui
- **状态管理**: Zustand
- **可视化**: ReactFlow (学习路径) + Mermaid (思维导图)
- **实时通信**: SSE (流式输出) + WebSocket (进度推送)
- **样式**: Tailwind CSS + shadcn/ui 组件库

## 4. 类型定义

### 4.1 核心类型 (`src/types/index.ts`)

```typescript
// 学生信息
export interface Student {
  id: string;
  student_no: string;
  name: string;
  email: string;
  avatar_url?: string;
  grade?: string;
  major?: string;
}

// 学生画像
export interface StudentProfile {
  id: string;
  student_id: string;
  course_id: string;
  dimensions: ProfileDimensions;
  confidence: number;
  version: number;
}

// 画像六维
export interface ProfileDimensions {
  knowledge_mastery: Record<string, number>;
  learning_style: Record<string, number>;
  cognitive_level: Record<string, number>;
  interest_preferences: Record<string, number>;
  weak_topics: string[];
  learning_pace: {
    daily_hours: number;
    preferred_time: string;
    focus_duration_min: number;
  };
}

// 学习资源
export interface Resource {
  id: string;
  type: "explanation" | "mindmap" | "exercise" | "code" | "audio" | "path" | "qa";
  title: string;
  content?: string;
  mermaid_code?: string;
  code_snippets?: CodeSnippet[];
  audio_url?: string;
  exercises?: Exercise[];
  source_refs?: SourceRef[];
  is_verified: boolean;
  created_at: string;
}

// 代码片段
export interface CodeSnippet {
  language: string;
  code: string;
  explanation?: string;
}

// 练习题
export interface Exercise {
  type: "choice" | "fill" | "code" | "short_answer";
  difficulty: "easy" | "medium" | "hard";
  question: string;
  options?: { key: string; text: string; is_correct: boolean }[];
  answer: string;
  explanation?: string;
  sample_code?: string;
}

// 来源引用
export interface SourceRef {
  file: string;
  page: number;
  section?: string;
  similarity: number;
}

// 学习路径
export interface LearningPath {
  id: string;
  name: string;
  description?: string;
  nodes: PathNode[];
  edges: PathEdge[];
  daily_plan?: DailyPlan[];
  progress: number;
  total_hours: number;
}

// 路径节点
export interface PathNode {
  id: string;
  kp_name: string;
  order: number;
  difficulty: string;
  status: "pending" | "in_progress" | "completed";
  estimated_hours: number;
  priority: string;
  category: string;
}

// 路径边
export interface PathEdge {
  from: string;
  to: string;
  relation: string;
}

// 每日计划
export interface DailyPlan {
  day: number;
  items: { kp_name: string; estimated_hours: number; priority: string }[];
  total_hours: number;
}

// 任务进度
export interface TaskProgress {
  task_id: string;
  status: "pending" | "running" | "completed" | "failed";
  progress: number;
  current_step: string;
  message?: string;
}

// 聊天消息
export interface ChatMessage {
  id: string;
  role: "user" | "assistant" | "system";
  content: string;
  resources?: Resource[];
  citations?: string[];
  timestamp: string;
}

// SSE 事件
export interface SSEEvent {
  event: "start" | "progress" | "resource" | "message" | "done" | "error";
  data: string;
}

// API 响应
export interface ApiResponse<T> {
  data: T;
  message?: string;
  success: boolean;
}

// 分页响应
export interface PaginatedResponse<T> {
  items: T[];
  total: number;
  page: number;
  page_size: number;
}
```

## 5. 组件开发计划

### 5.1 布局组件 (优先级: P0)

#### 5.1.1 MainLayout.tsx
- **功能**: 主布局容器，包含侧边栏和内容区域
- **依赖**: Sidebar, Header
- **状态**: 无

#### 5.1.2 Sidebar.tsx
- **功能**: 侧边栏导航，显示菜单项
- **菜单项**: 首页、对话、画像、资源、路径、练习
- **依赖**: lucide-react 图标
- **状态**: 当前选中项

#### 5.1.3 Header.tsx
- **功能**: 顶部导航栏，显示用户信息和操作
- **依赖**: 学生信息
- **状态**: 用户头像、通知

### 5.2 对话组件 (优先级: P0)

#### 5.2.1 ChatWindow.tsx
- **功能**: 对话窗口，支持 SSE 流式输出
- **核心逻辑**:
  - 使用 EventSource 接收 SSE 事件
  - 流式渲染消息
  - 显示任务进度
  - 资源卡片展示
- **依赖**: MessageBubble, ResourceCard, useChatStore
- **状态**: 消息列表、加载状态、进度

#### 5.2.2 MessageBubble.tsx
- **功能**: 消息气泡，区分用户和助手消息
- **特性**: 
  - 支持 Markdown 渲染
  - 代码高亮
  - 引用标注
- **依赖**: react-markdown, react-syntax-highlighter

#### 5.2.3 ResourceCard.tsx
- **功能**: 资源卡片，根据类型显示不同内容
- **类型**: 讲解、思维导图、练习、代码、音频、路径、问答
- **特性**: 
  - 可展开/收起
  - 来源引用
  - 验证状态
- **依赖**: MindMapCard (思维导图类型)

#### 5.2.4 ChatInput.tsx
- **功能**: 输入框组件
- **特性**:
  - 支持 Enter 发送
  - Shift+Enter 换行
  - 发送按钮状态
  - 快捷操作按钮

### 5.3 画像组件 (优先级: P0)

#### 5.3.1 ProfileRadar.tsx
- **功能**: 六维雷达图，可视化学生画像
- **维度**: 知识掌握、学习风格、认知水平、兴趣偏好、薄弱环节、学习节奏
- **依赖**: 自定义 SVG 或 recharts
- **状态**: 画像数据

#### 5.3.2 Questionnaire.tsx
- **功能**: 画像问卷表单
- **问题类型**: 单选、多选、滑块、文本
- **特性**:
  - 分步骤填写
  - 实时验证
  - 提交后调用 API
- **依赖**: react-hook-form, zod

#### 5.3.3 ProfileCard.tsx
- **功能**: 画像摘要卡片
- **显示**: 置信度、版本、更新时间
- **操作**: 查看详情、更新画像

### 5.4 可视化组件 (优先级: P1)

#### 5.4.1 MindMapCard.tsx
- **功能**: 思维导图渲染 (Mermaid)
- **特性**:
  - 动态渲染 Mermaid 代码
  - 缩放控制
  - 下载 SVG
  - 全屏查看
- **依赖**: mermaid 库
- **注意**: 必须使用 `'use client'` + `dynamic(..., { ssr: false })`

#### 5.4.2 PathVisualization.tsx
- **功能**: 学习路径可视化 (ReactFlow)
- **特性**:
  - 自定义节点样式
  - 状态颜色区分
  - 连线动画
  - 缩放和平移
  - 迷你地图
- **依赖**: reactflow 库
- **注意**: 必须使用 `'use client'` + `dynamic(..., { ssr: false })`

#### 5.4.3 DailyPlan.tsx
- **功能**: 每日学习计划展示
- **显示**: 日期、学习项、预计时长
- **交互**: 点击查看详情

### 5.5 练习组件 (优先级: P1)

#### 5.5.1 ExerciseCard.tsx
- **功能**: 练习题卡片
- **题型**: 选择题、填空题、编程题、简答题
- **特性**:
  - 难度标识
  - 选项交互
  - 提交答案
  - 查看解析

#### 5.5.2 ExerciseResult.tsx
- **功能**: 练习结果展示
- **显示**: 得分、正确率、用时
- **操作**: 重新练习、查看解析

## 12. 已知问题与解决方案

### 12.1 ReactFlow/Mermaid SSR 问题

**问题**: App Router 编译失败  
**解决**: 
```typescript
"use client";
import dynamic from "next/dynamic";

const PathVisualization = dynamic(
  () => import("@/components/path/PathVisualization"),
  { ssr: false }
);
```

### 12.2 SSE 连接管理

**问题**: 页面切换时连接未清理  
**解决**: 
```typescript
useEffect(() => {
  return () => {
    eventSource?.close();
  };
}, []);
```

### 12.3 状态持久化

**问题**: 页面刷新后状态丢失  
**解决**: 使用 Zustand persist 中间件
