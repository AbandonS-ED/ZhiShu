# 智枢(SmartHub) 前端开发进度

## 技术栈

- **框架**: Next.js 14.2.5 (App Router)
- **UI库**: React 18.3.1 + TypeScript 5.5.3
- **样式**: Tailwind CSS 3.4.6 + shadcn/ui (Radix UI)
- **状态管理**: Zustand 4.5.4
- **图表**: Recharts 2.12.7 + ReactFlow 11.11.4
- **数据获取**: SWR 2.2.5
