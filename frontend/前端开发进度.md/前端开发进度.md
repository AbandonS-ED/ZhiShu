# 智枢(SmartHub) 前端开发进度

最后更新：2026-06-07

## 技术栈

- **框架**: Next.js 14.2.5 (App Router)
- **样式**: Tailwind CSS 3.4.6 + 自定义 CSS（模板 1:1 复刻，无 shadcn/ui）
- **状态管理**: React useState（各页面独立管理）
- **依赖**: lucide-react（图标）, zustand/swr（待后端接入后使用）
