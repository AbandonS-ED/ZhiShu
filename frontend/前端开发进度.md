## 关键设计决策

### 1. 布局隔离（NO_SHELL_ROUTES）
根 `app/layout.tsx` 维护 `NO_SHELL_ROUTES = ['/login', '/admin']`，让管理后台完全跳过学生端 Sidebar/Header，独立布局。

### 2. Token 隔离
- 学生端：`zhishu_token` / `zhishu_student` / `zhishu_refresh_token`
- 管理端：`zhishu_admin_token` / `zhishu_admin_user`
- 两套互不干扰，避免误操作

### 3. 批量删除模式
所有 6 个列表页统一模式：
```ts
const sel = useSelection(filtered)
function batchDelete() {
  if (sel.selectedCount === 0) return
  if (!confirm(`确认删除选中的 ${sel.selectedCount} 个 X？`)) return
  const ids = Array.from(sel.selected)
  setList(prev => prev.filter(x => !ids.includes(x.id)))
  sel.clear()
}
```

### 4. CSS 命名空间
- 学生端：`.card` / `.chat-main` / `.stat` / `.btn`
- 管理端：`.admin-cd` / `.admin-cb` / `.admin-batch-bar` / `.admin-btn`
- 用 `admin-` 前缀完全隔离，避免样式冲突
