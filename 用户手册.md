## 1. 快速开始

### 1.1 注册与登录

1. 访问系统首页
2. 点击"注册"，填写：
   - 姓名
   - 邮箱
   - 密码
   - 专业
   - 年级
3. 注册成功后自动登录

### 1.2 首次使用

首次登录后，系统会引导您：
1. 构建学习画像（对话式）
2. 选择目标课程
3. 进入学习主界面

---

## 2. 学习画像

### 2.1 构建画像

进入"学习画像"页面，通过对话构建您的学习画像：

**示例对话**：
```
您：我是计算机专业大三学生，想考研
系统：好的，我了解了您的基本情况。请问您之前学过哪些相关课程？
您：学过数据结构、线性代数、概率论
系统：您更喜欢通过什么方式学习？看视频、读文档还是做练习？
您：我喜欢看视频和做练习
系统：您每天大概能投入多少时间学习？
您：大概3小时
```

系统会自动提取6个维度的特征：
- **知识基础**：您已掌握的知识
- **认知风格**：您的学习偏好
- **学习目标**：您的学习目的
- **易错点**：您的薄弱环节
- **学习节奏**：您的时间安排
- **兴趣方向**：您感兴趣的主题

### 2.2 查看画像

在"学习画像"页面可以查看：
- 6维度雷达图
- 知识点掌握度
- 画像完整度

### 2.3 画像更新

画像会随您的学习行为自动更新：
- 完成练习题 → 更新知识掌握度
- 收藏资源 → 更新兴趣方向
- 学习时长 → 更新学习节奏

## 3. 资源生成

### 3.1 生成学习资源

在对话窗口中描述您的需求：

**示例**：
```
您：帮我生成神经网络的学习资料
系统：好的，我将为您生成以下资源：
      - 学习文档
      - 思维导图
      - 练习题
      - 代码示例
      正在生成中...
```

系统会并行生成5种类型的资源。

### 3.2 资源类型说明

| 类型 | 说明 | 用途 |
|------|------|------|
| 学习文档 | Markdown格式的讲解文档 | 系统学习知识点 |
| 思维导图 | 可视化知识结构 | 理解知识体系 |
| 练习题 | 多类型题目（选择/填空/编程） | 检验学习效果 |
| 代码示例 | 带注释的代码 | 动手实践 |
| 视频脚本 | 教学视频脚本+音频 | 多模态学习 |

### 3.3 查看资源

在"资源中心"页面可以：
- 按类型筛选资源
- 按知识点搜索资源
- 查看资源详情
- 收藏喜欢的资源

## 4. 学习路径

### 4.1 生成学习路径

在对话窗口中请求：
```
您：帮我生成人工智能导论的学习路径
```

系统会根据您的画像生成个性化路径。

### 4.2 查看路径

在"学习路径"页面可以查看：
- DAG可视化图
- 知识点顺序
- 预计学习时长
- 当前进度

### 4.3 路径节点状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 已完成 | 绿色 | 已掌握的知识点 |
| 进行中 | 蓝色 | 当前正在学习 |
| 未开始 | 灰色 | 尚未学习 |

### 4.4 路径更新

路径会根据学习进度动态更新：
- 完成某个知识点 → 自动解锁下一个
- 练习正确率低 → 建议复习

## 5. 智能辅导

### 5.1 提问

在对话窗口中直接提问：

**示例**：
```
您：什么是梯度下降？
系统：梯度下降是一种优化算法，用于最小化损失函数。
      基本思想是沿着损失函数梯度的反方向更新参数...
      
      [1] 机器学习教材 第5章
      [2] 深度学习基础 PPT
```

### 5.2 上下文感知

系统知道您当前正在学习什么，回答会结合上下文：

```
您：这个公式怎么理解？（当前正在学习神经网络）
系统：您提到的"这个公式"是指神经网络中的反向传播公式吗？
      让我结合您正在学习的内容来解释...
```

### 5.3 引用源

所有回答都会标注引用来源：
- 知识库中的教材内容
- PPT内容
- 习题集内容

## 7. 常见问题

### Q1: 画像不准确怎么办？

A: 您可以通过对话补充信息，系统会自动更新画像。

### Q2: 生成的资源质量不好怎么办？

A: 可以重新请求生成，或在对话中提出具体要求。

### Q3: 学习路径不合理怎么办？

A: 可以在对话中说明您的情况，系统会重新规划。

### Q4: 回答不准确怎么办？

A: 系统基于知识库回答，如果知识库中没有相关内容，会明确告知。

### Q5: 如何提高画像完整度？

A: 多与系统对话，完成更多练习，系统会自动完善画像。
